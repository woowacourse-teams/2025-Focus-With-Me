# ✈️🧑‍🤝‍🧑 튜립 (Turip)
> 유튜버의 여행 정보를 한눈에 빠르게 확인할 수 있는 서비스

## 📝 프로젝트 소개
> 이 서비스는 유튜버가 실제 다녀온 여행 동선, 장소, 후기, 영상 등을 수집하고 정리해서 제공하는 앱입니다.  
사용자는 지역을 기준으로 여행 루트를 검색할 수 있습니다.
